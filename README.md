## ğŸ“ Deskripsi Program

Program ini merupakan aplikasi **Sistem Manajemen Tugas Mata Kuliah** yang dibuat untuk memenuhi Tugas Besar mata kuliah Struktur Data. Program ini membantu pengguna dalam mengelola data mata kuliah beserta daftar tugas yang dimiliki oleh setiap mata kuliah.

Setiap mata kuliah disimpan menggunakan **Binary Search Tree (BST)** berdasarkan kode mata kuliah, sedangkan tugas-tugas pada masing-masing mata kuliah disimpan menggunakan **Multi Linked List (MLL)** dengan relasi satu ke banyak (1-N).

---

## ğŸ§© Struktur Data yang Digunakan

1. **Binary Search Tree (BST)**
   Digunakan untuk menyimpan data mata kuliah agar proses pencarian, penambahan, dan pengurutan data dapat dilakukan dengan lebih efisien berdasarkan kode mata kuliah.

2. **Multi Linked List (MLL)**
   Digunakan untuk menyimpan daftar tugas pada setiap mata kuliah, di mana satu mata kuliah dapat memiliki lebih dari satu tugas.

3. **Array Tipe Dasar**
   Digunakan pada proses pengolahan data sederhana seperti menu pilihan dan perhitungan jumlah data sebagai penerapan tipe data dasar.

## ğŸ”„ Alur Kerja Program

1. Pengguna memasukkan data mata kuliah yang akan disimpan ke dalam BST.
2. Setiap mata kuliah dapat ditambahkan beberapa tugas yang tersimpan dalam linked list.
3. Pengguna dapat mencari mata kuliah berdasarkan kode.
4. Sistem menampilkan detail mata kuliah beserta seluruh tugas yang dimiliki.
5. Program menyediakan fitur pengurutan data secara inorder, perhitungan jumlah mata kuliah, serta pencarian kode mata kuliah terkecil dan terbesar.
